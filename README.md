# Maneesha Prasanna — Portfolio

Modern personal portfolio built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Sky Background Video (optional)

For a cinematic aldemy-style hero, add a looping night-sky MP4 to `public/sky-bg.mp4`.
The site automatically uses it when present; otherwise an animated canvas sky is shown.

Free sources: [Pexels](https://www.pexels.com/search/videos/night%20sky/), [Mixkit](https://mixkit.co/free-stock-video/night/).

## Deploy

Deploy easily on [Vercel](https://vercel.com) or any platform supporting Next.js.
